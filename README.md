# Brainlab

1. 저장소 디렉토리로 이동

<pre>
<code>
cd ~/경로
</code>
</pre>

2. 변경 사항 확인

<pre>
<code>
git status
</code>
</pre>

3. 수정 혹은 파일 올리기

<pre>
<code>
git add 파일명 # 수정할 파일 혹은 올릴 파일
git add . # 로컬 저장소 전체 올리기 혹은 수정
</code>
</pre>

4. 커밋

<pre>
<code>
git commit -m "내용"
</code>
</pre>

5. Github push

<pre>
<code>
git push
</code>
</pre>
